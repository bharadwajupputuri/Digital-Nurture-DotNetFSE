# Employee Management System – Scalar Function in MySQL

## Exercise 7: Return Data from a Scalar Function

---

## Objective

The objective of this exercise is to understand and implement a **Scalar Function** in MySQL. The function is created to calculate and return the annual salary of a specific employee using the employee's monthly salary. This exercise demonstrates how functions accept parameters, perform calculations, and return a single value.

---

## Software Requirements

- MySQL Server 8.0 or above
- MySQL Workbench
- Windows Operating System
